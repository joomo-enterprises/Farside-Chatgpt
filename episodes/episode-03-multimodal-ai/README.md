## Description

Go beyond text! This episode dives into multimodal AI — models that understand images, audio, and video alongside text. Learn how to build applications that combine vision and language, transcribe and understand audio, and work with the latest generative models for images and video.

- Python 3.10+

- Samples images (provided in the episode assets)

## How to Run

```bash
# Install dependencies
pip install -r requirements.txt

# Set your API key
export OPENAI_API_KEY=*** # Run the examples
```
