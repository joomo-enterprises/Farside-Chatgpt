## Description

A head-to-head comparison of the most popular large language models. We put GPT-4o, Claude Sonnet, Gemini, and Llama through a series of benchmarks and real-world tasks to see how they stack up. Learn the strengths and weaknesses of each model and when to reach for which one.

## Prerequisites

- Python 3.10+
- API keys for OpenAI, Anthropic, and Google (set as environment variables)
- Familiarity with basic LLM API calls

## How to Run

```bash
# Install dependencies
pip install -r requirements.txt

# Set your API keys
export OPENAI_API_KEY=export ANTHROPIC_API_KEY=*** export GOOGLE_API_KEY=*** # Run the comparison
cd src/
python benchmark.py
```
