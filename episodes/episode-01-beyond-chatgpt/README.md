## Description

An introduction to the AI landscape beyond ChatGPT. This episode explores the broader ecosystem of large language models, what makes them different, and how to think about choosing the right tool for the job. We cover the major players, open-source alternatives, and the key concepts you need to understand before diving deeper.

## Prerequisites

- Python 3.10+
- Basic understanding of Python
- An OpenAI API key (set as `OPENAI_API_KEY` environment variable)

## How to Run

```bash
# Install dependencies
pip install -r requirements.txt

# Set your API key
export OPENAI_API_KEY="sk-..."

# Run the examples
cd src/
python main.py
```

## What You'll Learn

- The difference between various LLM providers
- How to make API calls to different model providers
- When to use which model for different tasks
- The trade-offs between open-source and proprietary models
